4.3.2. Gew√∂hnliche DGL#

Bei gew√∂hnlichen Differentialgleichungen werden zwei F√§lle unterschieden: Anfangswertproblem und Randwertproblem. F√ºr beide gilt, dass die L√∂sung auf einem Intervall gesucht wird, jedoch wird im ersten Fall nur der Anfangswert vorgegeben und im zweiten Fall die L√∂sungswerte an den (im eindimensionalen Fall zwei) Randpunkten des Intervalls.

In diesem Kapitel wird nur auf das Anfangswertproblem eingegangen. Als Beispiel daf√ºr und dessen L√∂sung werden im Folgenden zwei Gleichungen vorgestellt und numerisch gel√∂st.

Hinweis: Alle physikalischen Konstanten sind in diesem Kapitel vernachl√§ssigt, um den Fokus auf die numerischen L√∂sungsmethoden zu setzten.

Temperaturentwicklung#

Im ersten Beispiel wird die zeitliche Temperaturentwicklung \(\sf T(t)\) mittels der Gleichung

\[\sf T'(t) = -(T(t)-T_\infty) + Q(t)\]

vorgeschrieben und ein Anfangswert, d.h. \(\sf T(t=0) = T_0\), gegeben. Dabei ist \(\sf T_\infty\) eine Konstante und \(\sf Q(t)\) eine Funktion der Zeit. Die Gleichung beschreibt die √Ñnderung der Temperatur aufgrund des Unterschieds zur Umgebung (\(\sf T_\infty\)) und eines vorgegebenen W√§rmestroms (\(\sf Q(t)\)).

Die Diskretisierung in der Zeit mit einem frei gew√§hlten \(\sf \Delta t\) f√ºhrt zu folgender N√§herung der Ableitung

\[\sf \frac{T^{n+1} - T^n}{\Delta t} = -(T(t)-T_\infty) + Q(t)\]

Hierbei wird der Index, welcher der Zeitpunkt angibt, oben aufgef√ºhrt. Dies ist eine Konvention, welche sp√§ter bei partiellen Differentialgleichungen die Indizierung lesbarer macht.

Die Terme auf der rechnten Seite der Gleichung sind aber noch nicht diskretisiert. Hierbei gibt es zwei M√∂glichkeiten: die Auswertung zum Zeitpunkt \(\sf n\) oder \(\sf n+1\). Wird die erste gew√§hlt, so f√ºhrt dies zum expliziten Euler-Verfahren

\[\sf \frac{T^{n+1} - T^n}{\Delta t} = -(T^n-T_\infty) + Q^n\]

Die zweite M√∂glichkeit, d.h. die Auswertung der Terme zum Zeitpunkt \(\sf n+1\) f√ºhrt zum impliziten Euler-Verfahren, auf welches hier nicht weiter eingegangen wird.

Die Umformung der Gleichung, welche nach dem expliziten Euler-Verfahren erzeugt wurde, nach dem n√§chsten Wert der L√∂sung, d.h. \(\sf T^{n+1}\), ergibt

\[ \sf T^{n+1} = T^n -\Delta t (T^n-T_\infty) + Q^n\]

Damit kann mittels der Informationen zum Zeitpunkt \(\sf n\) die L√∂sung zum Zeitpunkt \(\sf n+1\) direkt (explizit) berechnet werden. Mit dem vorgegebenen Anfangswert, \(\sf T^0 = T_0\) kann der n√§chste L√∂sungswert \(\sf T^1\), daraus dann \(\sf T^2\) und so weiter berechnet werden.

Als erstes L√∂sungsbeispiel wird

\[\sf T_0 = 400,\quad T_\infty = 300,\quad Q(t) = 0 \]

gew√§hlt und die L√∂sung soll bis \(\sf t=10\) mit \(\sf \Delta t=0.5\) gesucht werden.

# Setzten der Konstanten
T0 = 400
Tinfty = 300
dt = 0.5

# Anlegen des Zeit- und L√∂sungsarrays
zeit = np.arange(0, 10, dt)
T = np.zeros_like(zeit)

# Setzten des Anfangswerts
T[0] = T0

# L√∂sung nach dem expliziten Euler-Verfahren
for n in range(0, len(zeit)-1):
    T[n+1] = T[n] - dt * (T[n] - Tinfty)

# Visualisierung der L√∂sung
plt.plot(zeit, T, marker='.')
plt.xlabel("t")
plt.ylabel("T");

Im zweiten L√∂sungsbeispiel wird nun die Funktion \(\sf Q(t)\) als Stufenfunktion hinzugenommen, mit

\[\begin{split}\sf Q(t) = \left\{ \begin{array}{cc} 100 & 5 \le t \le 15 \\ 0 & \text{sonst} \end{array} \right. \end{split}\]

und die L√∂sung soll bis \(\sf t=25\) berechnet werden.

# Anlegen des Zeit- und L√∂sungsarrays
zeit = np.arange(0, 25, dt)
T = np.zeros_like(zeit)

# Setzten des Anfangswerts
T[0] = T0

# L√∂sung nach dem expliziten Euler-Verfahren
for n in range(0, len(zeit)-1):
    t = zeit[n]
    
    # Quellterm Q
    Q = 0
    if t >= 5 and t <= 15:
        Q = 100
    
    # Zeititeration
    T[n+1] = T[n] - dt * (T[n] - Tinfty) + Q

# Visualisierung der L√∂sung

plt.xlabel("t")
plt.ylabel("T");

Schwingung#

Genauso wie das vorangegangene Beispiel mit der Temperaturentwicklung, kann auch die Gleichung f√ºr eine ged√§mpfte harmonische Schwingung mit dem expliziten Euler-Verfahren gel√∂st werden. Die gew√∂hnliche Differentialgleichung zweiter Ordnung f√ºr die Auslenkung \(\sf u(t)\) lautet

\[ u''(t) + u'(t) + u(t) = 0 \]

Oder als System von zwei Gleichungen erster Ordnung

\[ u'(t) = v(t) \]
\[ v'(t) = -v(t) - u(t) \]

Die diskretisierte Form lautet

\[ u^{n+1} = u^n + \Delta t v^n \]
\[ v^{n+1} = v^n - \Delta t(v^n + u^n) \]

Die L√∂sung soll mit dem Anfangswerten \(\sf u(t=0) = u_0 = 1\) und \(\sf v(t=0) = v_0 = 0\) bis zur Zeit \(\sf t=25\) mit \(\sf \Delta t = 0.25\) berechnet werden.

# Setzten der Konstanten
u0 = 1
v0 = 0
dt = 0.25

# Anlegen des Zeit- und L√∂sungsarrays
zeit = np.arange(0, 25, dt)
u = np.zeros_like(zeit)
v = np.zeros_like(zeit)

# Setzten der Anfangswerte
u[0] = u0
v[0] = v0

# L√∂sung nach dem expliziten Euler-Verfahren
for n in range(0, len(zeit)-1):
    u[n+1] = u[n] + dt * v[n]
    v[n+1] = v[n] - dt * (v[n] + u[n])

# Visualisierung der L√∂sung
plt.plot(zeit, u, marker='.')
plt.xlabel("Zeit t")
plt.ylabel("Amplitude u");

L√∂sung mit dem scipy-Modul#

Das oben vorgestellte L√∂sungsverfahren mit der expliziten Euler-Methode ist eines der einfachsten Verfahren. Es existieren viele, vor allem bessere, Verfahren zur L√∂sung von gew√∂hnlichen Differentialgleichungen, welche hier aber nicht weiter vorgestellt werden. Einige der h√§ufig eingesetzten L√∂ser, wie z.B. das Runge-Kutta-Verfahren sind bereits im scipy-Modul implementiert. Anhand der obigen Anfangswertprobleme, wird deren Verwendung demonstriert.

Die Form, in welcher die zu l√∂senden Differentialgleichungen erster Ordnung vorliegen m√ºssen ist

\[ y'(t) = f(y(t), t) \]

mit den Anfangsbedingungen f√ºr \(\sf y(t=0) = y_0\). Dabei wird f√ºr \(\sf f\) oft der Begriff ‚ÄòRechte-Hand-Seite‚Äô, englisch right hand side (RHS), verwendet. Wird ein System von Gleichungen gel√∂st, so sind \(\sf y\) und \(\sf f\) entsprechend Vektoren.

Mit dieser Darstellung lautet die Funktion \(\sf f\) f√ºr die Temperaturgleichung

\[ f(T(t), t) = -(T(t)-T_\infty) + Q(t)\]

Die Funktion scipy.integrate.solve_ivp ben√∂tigt folgende Informationen f√ºr die L√∂sung der Gleichung

  • die Funktion \(\sf f\)

  • das Intervall auf dem die Gleichung gel√∂st werden soll, d.h. hier \(\sf t \in [0, t_{ende}]\)

  • den Anfangswert \(\sf T_0\)

Die Funktion \(\sf f\) muss dabei als eine Python-Funktion implementiert werden, welche zu gegebenen \(\sf t\) und \(\sf T\) eine entsprechende R√ºckgabe liefert. Dabei muss das erste Argument die unabh√§ngige Variable sein, hier die Zeit \(\sf t\).

# Definition der Funktion f, d.h. der RHS
def f_T(t, T):
    Tinfty = 300
    Q = 0
    if t >= 5 and t <= 15: 
        Q = 100
    return -(T-Tinfty) + Q

# Definition der Konstanten
t_ende = 25
T0 = 400

Mit diesen Informationen kann nun der L√∂ser aufgerufen werden. Das dabei verwendetet L√∂sungsverfahren kann mit dem optionalen Argument method ausgew√§hlt werden, as Standardwert wird ein Runge-Kutta-Verfahren der f√ºnften Ordnung (RK45) verwendet.

# Aufruf des L√∂sers

Da es sich im Beispiel der Schwingungsgleichung von oben um ein Gleichungssystem handelt, ist die Funktion \(\sf \vec{f}\) folgende Vektorform an

\[\begin{split} \sf \vec{z}(t) = \left( \begin{array}{c} u(t)\\ v(t) \end{array} \right) \end{split}\]
\[\begin{split} \sf \vec{f}(\vec{z}(t), t) = \left( \begin{array}{c} v(t)\\ -v(t) - u(t) \end{array} \right) \end{split}\]

Die Implementierung sieht entsprechend aus:

def f_osz(t, z):
    u = z[0]
    v = z[1]
    # u' 
    up = v
    # v'
    vp = - v - u
    return [up, vp]
